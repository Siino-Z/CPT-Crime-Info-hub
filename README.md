# CPT-Crime-Info-hub

## Overview

**CPT-Crime-Info-hub** is an advanced crime intelligence dashboard for Cape Town, South Africa. It provides real-time analytics and interactive visualizations to help communities and indivisuals make informed decisions about safety and crime trends.

## Features

- Interactive map of crime incidents and hotspots
- Real-time and historical crime statistics
- Area selector for focused analysis
- Critical alerts and trend detection
- Community safety tips and emergency information
- Data sourced from SAPS and crime intelligence analysis

## Technologies Used

- HTML, CSS, JavaScript
- [Leaflet.js](https://leafletjs.com/) for interactive maps
- [PapaParse](https://www.papaparse.com/) for CSV data parsing

## Project Information

- Data was sourced from the South African Police Service (SAPS) and crime intelligence analysis.
- For demonstration and educational purposes.
